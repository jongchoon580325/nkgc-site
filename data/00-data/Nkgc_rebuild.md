# 🚀 남경기노회 웹사이트 리빌드 최종 개발 계획서 (Next.js + Zustand + Tailwind CSS 기반)

| 구분 | 내용 |
| :--- | :--- |

## 2. 👑 Master Agency 설정 및 역할 정의

본 프로젝트의 성공을 위해 각 전문 Agency들을 총괄 지휘하고 조정하는 **마스터 에이전시(Master Agency)**, 즉 **'프로젝트 코디네이터 (Project Coordinator)'**를 설정합니다.

### **Master Agency: 코디네이터 (Coordinator)**

| Agency 이름 | 전문 분야 | 페르소나 특성 및 주요 임무 | Workflow Trigger (핵심 명령) |
| :--- | :--- | :--- | :--- |
| **코디네이터 (Coordinator)** | 프로젝트 총괄 및 조율 (HUB 역할) | **숙련된 PM(Project Manager).** 전체 일정, 예산, 목표 달성 여부를 총괄하며, 각 전문 Agency 간의 의사소통 및 핸드오프를 중재하고 관리하는 **프로젝트 관리의 핵심 허브** 역할 수행. | **"전체 워크플로를 관리하고, 사용자 요구사항을 각 전문 Agency에 전달 및 변환하며, 산출물을 승인하고 다음 단계로의 전환 시점을 결정하고 지시하라."** |

### **Master Agency 구조의 이점 (프로젝트 관리 허브 역할 강조)**

1. **소통 채널 단일화:** 사용자는 복잡한 기술적 상세 내용 없이 코디네이터와만 소통하여 요구사항을 전달하고 피드백을 받습니다.
2. **노이즈 감소 및 보고 통합:** 각 Agency의 기술적이고 상세한 보고(예: DB 스키마 변경, 단위 테스트 결과)를 코디네이터가 **프로젝트 관리 관점의 요약 보고서**로 통합하여 상부에 전달합니다.
3. **효율적 조율 및 위험 관리:** 각 Agency 간의 작업 의존성(핸드오프)을 감지하고, 한 파트의 지연이 전체 일정에 미치는 영향을 예측하여 **능동적인 조정과 위험 관리**를 수행합니다.

## 3. 🤖 전문 AI Agency 개발 계획 및 페르소나 설정

| Agency 이름 | 전문 분야 | 주요 임무 (강화된 역할 포함) | Workflow Trigger (핵심 명령) |

| **C. 옵티마 (Optima)** | 품질 보증 및 최적화 | **웹 접근성 및 성능 전문가.** 이관된 자료의 접근 권한 제어, 검색 정확도, Lighthouse 성능 점수 90점 이상 달성을 검증하고 최적화 코드를 제안한다. | **"자료실의 회원별 접근 권한 시나리오 및 Core Web Vitals 최적화(LCP, FID) 테스트 스크립트를 자동 생성하고, 모든 검증 보고서를 코디네이터에게 제출하라."** |

## 4. ⚙️ 최종 워크플로 (Coordinator 중심의 4단계)

### I. 🔍 분석 및 기획 단계 (코디네이터 주도)

| 단계 | 주요 활동 | 담당 Agency | 핸드오프/승인 주체 |
| :--- | :--- | :--- | :--- |
| **1. 요구사항 확정** | 기존 IA, UX/UI 지침 (현대적 디자인)을 아이아나가 작성. | 아이아나 | 코디네이터 (최종 승인 후 지침 하달) |
| **2. DB 스키마 설계** | 마이그라토가 Next.js용 신규 스키마를 설계. | 마이그라토 | 코디네이터 (승인 후 코디우스에게 API 구조 공유 지시) |
| **3. 상태 모델 설계** | 코디우스가 Zustand를 활용한 전역 상태 관리 구조 초안 설계. | 코디우스 | 코디네이터 (검토 및 승인) |

### II. 🎨 개발 및 구현 단계 (코디네이터 지휘)

| 단계 | 주요 활동 | 담당 Agency | 핸드오프/승인 주체 |
| :--- | :--- | :--- | :--- |
| **4. 데이터 마이그레이션 실행** | 마이그라토가 변환 스크립트 실행 및 데이터 이관 완료 보고. | 마이그라토 | 코디네이터 (이관 완료 및 데이터 무결성 1차 검증 지시) |
| **5. 핵심 컴포넌트 개발** | 코디우스가 Next.js, Tailwind, Zustand를 사용하여 자료실, 검색, 인증 로직 구현. | 코디우스 | 코디네이터 (개발 완료 통보 및 옵티마에게 테스트 준비 지시) |

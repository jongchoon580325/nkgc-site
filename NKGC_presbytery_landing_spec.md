# 남경기노회 공식 랜딩페이지 재개발 프로젝트 명세서

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트 명** | 남경기노회 공식 랜딩페이지 (Homepage) 재개발 |
| **목표** | 노회원들의 주요 행정/정보 접근성 및 가독성 향상 |
| **적용 범위** | 메인 랜딩페이지 (/) |
| **구현 환경** | Next.js (SSR/SSG), TypeScript (선택), Styling Solution |

---

## 기술 스택

| 분류 | 기술 스택 | 설명 |
|------|-----------|------|
| **프론트엔드** | Next.js (App Router 권장) | SEO 최적화 및 고성능 제공 |
| **스타일링** | Tailwind CSS (권장) 또는 Styled-Components | 빠른 개발 및 반응형 디자인(Responsive Design) 용이성 확보 |
| **상태 관리** | React Context 또는 Recoil/Zustand (필요시) | 전역 상태 관리 (예: 로그인 상태, 테마 등) |
| **데이터 페칭** | SWR 또는 React Query (Next.js 서버 컴포넌트 활용 고려) | 노회 공지, 자료 목록 등의 데이터 비동기 처리 |

---

## 컴포넌트 구조

```
graph TD
    A[Layout Component] --> B(Header/Navigation)
    A --> C(Main Content Area)
    C --> D(Hero Section : 슬로건 & 메인 CTA)
    C --> E(Quick Actions : 핵심 행정 모듈)
    C --> F(Notice & News : 최신 공지/소식)
    C --> G(Gallery & Event Highlights : 사진/영상)
    C --> H(Affiliate Links : 관련 기관)
    A --> I(Footer)
```

## 주요 섹션 명세

### 1. Header/Navigation

| 요소 | 내용 | 비고 |
|------|------|------|
| **로고** | 노회 로고 및 명칭 (남경기노회) | 좌측 상단 고정, 클릭 시 Home 이동 |
| **메인 메뉴** | Home / 노회소개 / 행정 & 자료 / 공지 & 소통 | 4~5개로 간소화 (드롭다운 최소화) |
| **액션 버튼** | 로그인 / 회원가입 / 상회비 납부 (강조) | 우측에 배치, 로그인 상태에 따라 변경 |
| **UX/UI** | Sticky Header 적용, 스크롤 시 배경색 변경 또는 투명도 조절 | |

### 2. Hero Section

| 요소 | 내용 | 비고 |
|------|------|------|
| **배경** | 고화질의 노회 활동 사진 또는 짧은 비디오 루프 | 낮은 명도의 오버레이를 적용하여 텍스트 가독성 확보 |
| **슬로건** | "Coram Deo, Sola Scriptura." 또는 노회의 비전 문구 | H1 태그 사용 |
| **서브 텍스트** | "대한예수교장로회 남경기노회" | |
| **메인 CTA** | "노회 행정 서식 다운로드" 또는 "최신 노회 공지 확인" | Primary Color를 사용하여 가장 눈에 띄게 강조 |

### 3. Quick Actions
핵심 행정 모듈에 대한 빠른 접근 제공

### 4. Notice & News
최신 공지사항 및 노회 소식 표시

### 5. Gallery & Event Highlights
주요 활동 사진 및 영상 콘텐츠 전시

### 6. Affiliate Links
관련 기관 및 협력 단체 링크

### 7. Footer
연락처, 저작권 정보, 추가 링크 등

## 구현 조건 및 요구사항

| 항목 | 구현 조건 |
|------|-----------|
| **반응형 디자인** | Mobile-First로 설계하며, 모든 컴포넌트는 모바일(360px) 환경에서 완벽하게 작동해야 합니다. |
| **가독성** | 본문 폰트는 16px 이상, 명확한 타이포그래피(고딕 계열) 사용. 콘텐츠와 배경의 명도 대비를 확보해야 합니다. |
| **브랜딩** | 노회 공식 컬러(Primary/Accent)를 지정하여 CTA 및 아이콘에 통일성 있게 적용합니다. |
| **접근성** | 이미지에 alt 태그 필수 적용, 키보드 네비게이션이 가능하도록 ARIA 속성을 고려합니다. |
| **성능** | Next.js의 이미지 최적화 (next/image) 및 코드 스플리팅을 활용하여 Lighthouse 점수 90점 이상을 목표로 합니다. |

## 개발 우선순위

1. **1단계**: 프로젝트 초기 설정 및 기술 스택 구성
2. **2단계**: Layout 컴포넌트 및 Header/Footer 구현
3. **3단계**: Hero Section 및 주요 콘텐츠 섹션 개발
4. **4단계**: 반응형 디자인 및 접근성 최적화
5. **5단계**: 성능 테스트 및 SEO 최적화
6. **6단계**: 최종 QA 및 배포

# 관리자 메뉴 재구성 작업 (Admin Menu Restructuring)

## 목표

관리자 사이드바 메뉴를 직관적인 계층형 구조(Top Menu -> Sub Menu)로 변경하여 접근성과 편의성을 향상시킵니다.

## 작업 내용

### 1. 메뉴 구조 정의

기존의 평면적인 메뉴 목록을 다음과 같은 카테고리 기반 구조로 재정의합니다.

* **대시보드** (단일 메뉴)
* **노회행정**
  * 정회원관리
  * 상비부관리
  * 상회비관리
  * 별명부관리
* **노회자료**
  * 규칙관리
  * 결의서관리
  * 사진자료실 관리
  * 영상자료실 관리
  * 응시자 관리
* **노회소개**
  * 임원 관리
  * 콘텐츠 관리
* **기관소개**
  * 기관 관리
* **시찰소개**
  * 시찰 관리
* **노회알림 및 서식**
  * 게시판 설정
* **메인 페이지**
  * 히어로 관리
* **시스템 관리**
  * 데이터 관리

### 2. UI 컴포넌트 수정 (`AdminSidebar.tsx`)

* [x] `MenuItem` 인터페이스 확장 (서브메뉴 지원)
* [x] 메뉴 데이터 구조 변경 (카테고리별 그룹화)
* [x] 사이드바 렌더링 로직 수정
  * 카테고리 헤더 또는 아코디언 메뉴 구현
  * 사이드바 축소(`isCollapsed`) 시 동작 방식 처리 (아이콘만 표시하거나 호버 시 서브메뉴 표시 등)

### 3. 검증

* [x] 메뉴 클릭 시 해당 페이지로 정상 이동 확인
* [x] 사이드바 축소/확장 시 UI 깨짐 없는지 확인
* [x] 현재 활성화된 메뉴(Active State) 표시 확인

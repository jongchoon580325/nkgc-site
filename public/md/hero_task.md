# 히어로 섹션 완전 연동 작업 (Hero Section Complete Implementation)

## 목표

관리자가 홈 페이지 상단 히어로 섹션을 완전히 커스터마이징할 수 있도록 백엔드-프론트엔드를 연동합니다.

## 요구사항 정리

### 확정 사항

- **배경 스타일**: 완전 대체 방식 (이미지 업로드 시 기본 애니메이션 숨김)
- **CTA 버튼**: 고정 (수정 불가)
- **텍스트 커스터마이징**: 전부 수정 가능 (타이틀, 서브타이틀, 모토1~3, 설명)
- **미리보기**: 필요
- **모바일 배경**: PC용 이미지 자동 리사이즈

### 이미지 애니메이션 효과 (5가지)

1. `static` - 효과 없음 (정지 이미지)
2. `kenburns` - 느린 줌인/아웃 (30초 주기)
3. `wave` - 물결 왜곡 효과 (SVG 필터)
4. `pulse` - 미세한 맥동 (호흡하는 느낌)
5. `pan` - 느린 좌우 이동

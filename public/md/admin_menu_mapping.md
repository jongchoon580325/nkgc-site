# 관리자 메뉴 ↔ 공개 페이지 매핑 요약 (그룹별)

공개 페이지의 메뉴 구조를 기준으로, 관련된 관리자 페이지 메뉴들을 정리했습니다.

### 1. 노회행정

* **정회원관리** (노회원현황)
* **상비부관리** (상비부현황)
* **상회비관리** (상회비현황)
* **별명부관리** (별명부)

### 2. 노회자료

* **규칙관리** (규칙자료실)
* **결의서관리** (결의서자료실)
* **사진자료실 관리** (사진자료실)
* **영상자료실 관리** (영상자료실)
* **응시자 관리** (고시자료실)

### 3. 노회소개

* **임원 관리** (현재임원, 역대임원)
* **콘텐츠 관리** (노회장인사, 노회소개 등 정적 페이지)

### 4. 기관소개

* **기관 관리** (주교연합회, 남/여전도회, 학생/청장년면려회)

### 5. 시찰소개

* **시찰 관리** (시찰회 전체)

### 6. 노회알림 및 서식

* **게시판 설정** (노회공지, 회원게시판, 자유게시판, 행정서식, 자립위원회서식)

### 7. 메인 페이지 (Home)

* **히어로 관리** (상단 배너 이미지 및 텍스트)

### 8. 시스템 관리

* **데이터 관리** (백업, 복구, 초기화)
* **대시보드** (전체 현황 요약)
